# プライバシーポリシー

「セキュリティセルフチェック支援ツール」（以下、「本ツール」）は、個人開発者 slash-level（以下「開発者」）が提供するサービスです。
「セキュリティセルフチェック支援ツール」は、本ツールを利用するユーザー（以下、「ユーザー」）のプライバシーを尊重し、個人情報の適切な取扱いおよび保護に努めます。

本プライバシーポリシーは、本ツールにおける個人情報等の取扱いについて定めるものです。

### 1. 取得する情報

本ツールでは、以下の情報を取得します。

**(1) ユーザーが直接提供する情報**
*   メールアドレス（ユーザー登録を行った場合。ユーザーIDとして使用）
*   パスワード（ユーザー登録を行った場合。認証サービスにより安全に管理され、開発者が平文で取得することはありません）
*   匿名利用の場合、メールアドレスやパスワードは取得されません
*   評価セット、回答内容、備考、アクションアイテム等の入力データ
*   フィードバックフォーム（Google Forms）やお問い合わせを通じて、ユーザーが任意で提供する情報
    ※フィードバックフォームは本ツールとは別の外部サービスとして提供されます

**(2) 本ツールの利用に伴い自動的に取得される情報**
*   Firebase Authentication により発行されるユーザー識別情報（UID）
*   アクセス日時、操作履歴等のログ情報（サービスの安定運用および不正防止のため）

### 2. 利用目的

取得した情報は、以下の目的の範囲内で利用します。

*   ユーザー認証およびアカウント管理
*   評価セットの保存、表示、編集および削除
*   AIによる改善アドバイスの生成
*   ダッシュボード表示、レポート（PDF／CSV）の生成
*   本ツールの運用、保守、品質向上
*   不正利用の防止およびセキュリティ確保
*   利用規約違反への対応
*   ユーザーから提供されたフィードバック内容の分析および本ツールの改善


### 3. AIサービスの利用について

本ツールでは、AIによる改善アドバイス機能の提供のため、Google Gemini API を利用しています。

*   ユーザーが入力した評価内容および備考等は、AIアドバイス生成の目的で外部APIに送信される場合があります。
*   当該データの取扱いは、Google Gemini API の利用条件およびプライバシーポリシーに従って処理されます。
*   **※注意：** 秘密情報（顧客の実データ、パスワード、具体的なネットワーク構成情報等）の入力は控えてください。

### 4. 個人情報の管理

本ツールは、個人情報への不正アクセス、漏えい、改ざん、滅失等を防止するため、適切な技術的および組織的安全管理措置を講じます。

*   認証機能は Firebase Authentication を利用しています。
*   データベースはクラウドサービス（Supabase 等）上で管理されます。

### 5. 第三者提供について

法令に基づく場合を除き、ユーザーの個人情報を、ユーザーの同意なく第三者に提供することはありません。

ただし、以下の場合を除きます。

*   本ツールの提供に必要な範囲で、外部サービス（Firebase、Supabase、Google Gemini API、Google Forms 等）を利用する場合
*   法令に基づき開示を求められた場合

### 6. アカウント削除およびデータの消去

ユーザーは、本ツール上の機能を通じて、自身のアカウントを削除することができます。

アカウント削除が行われた場合、以下の情報は合理的な範囲で速やかに削除されます。

*   認証情報
*   評価セット、回答、アクションアイテム等の関連データ

### 7. クッキー（Cookie）等の使用について

本ツールでは、ログイン状態の維持やセキュリティ確保のため、認証に関連する技術（トークン等）を使用する場合があります。

これらは、本ツールの機能提供に必要な範囲でのみ使用されます。

### 8. 本プライバシーポリシーの変更

本プライバシーポリシーの内容は、法令の改正や本ツールの機能変更等に応じて、予告なく変更されることがあります。

変更後のプライバシーポリシーは、本ツール上に掲載された時点で効力を生じます。

### 9. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、本ツール内または公開されている連絡先を通じてご連絡ください。
